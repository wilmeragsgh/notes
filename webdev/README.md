# Web Development

### Updating node

```javascript
sudo npm cache clean -f
```

```javascript
sudo npm install -g n
```

```javascript
sudo n stable
```

## Setting a specific version of node

```
nvm use <version>
```

[reference](https://github.com/nvm-sh/nvm)
